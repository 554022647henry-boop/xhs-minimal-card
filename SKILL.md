# xhs-minimal-card · 极简笔记卡片

白底纯文字风格，适合长文内容卡片。

## 使用流程

收到用户请求后，按以下步骤执行：

### 第一步：收集必要信息

如果用户没有提供以下信息，**必须先询问**：

1. **账号名称**：顶部水印显示的账号名，例如「商分账号」「王小明」
2. **内容**：要生成卡片的文字内容（如果还没提供）
3. **输出目录**（可选）：默认输出到当前目录下的 `output/` 文件夹

示例提问：「你的小红书账号名称是什么？」

### 第二步：准备内容文件

将用户提供的内容保存为 `.txt` 文件，**严格保持原文**，不要修改用户的任何措辞。

用 `---` 分隔多张卡片（单独一行）。支持 `**加粗**` 语法。

### 第三步：运行渲染脚本

```bash
python scripts/render.py <内容文件> -b <账号名称> -o <输出目录>
```

示例：
```bash
python scripts/render.py content.txt -b 商分账号 -o ./output
```

### 第四步：告知结果

报告生成了几张卡片，输出路径在哪里。

---

## 脚本参数

| 参数 | 说明 | 默认 |
|------|------|------|
| `-b` | 顶部账号名水印 | 空 |
| `-o` | 输出目录 | 当前目录 |

## 依赖

```bash
pip install playwright
playwright install chromium
```
